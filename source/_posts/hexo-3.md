> 以下高频命令，总结一下

<!--more-->

### hexo
> 初始化博客
> ```base
> haxe init _name_
> ```
> 清空博客缓存
> ```base
> haxe clean
> ```
> 生成博客静态文件
> ```base
> haxe g
> ```

### git
> 添加到缓冲区
> ```base
> git add *
> ```
> 提交
> ```base
> git commit -m '______'
> ```
> 推送到远程仓库
> ```base
> git push
> ```
